# NutriFit Advisor

## Descrição

O NutriFit Advisor é uma aplicação desenvolvida em Python com o objetivo de calcular o Índice de Massa Corporal (IMC) do utilizador e fornecer sugestões de suplementação de acordo com o seu perfil físico e objetivo pessoal.

Este projeto foi desenvolvido no âmbito da unidade curricular **10790 - Projeto de Programação**.

---

## Funcionalidades

- Recolha de dados do utilizador
- Cálculo automático do IMC
- Classificação do IMC
- Definição de objetivo físico
- Sugestão de suplementação
- Geração de relatório final
